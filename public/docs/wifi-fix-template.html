Quick Fix Guide

Fix Wi‑Fi connection issues

A one‑pager you can hand to users when they can’t get online.

Context

When to use this

For everyday Wi‑Fi hiccups: home or office users who suddenly can’t connect, but the router has lights and other devices may still work.

Simple fix

Forget & reconnect (Windows)

  1. Click the Wi‑Fi icon » right‑click your network » Forget.
  2. Toggle Airplane mode on, wait 5 seconds, toggle off.
  3. Select your network again, enter the password, and connect.
  4. If it still fails: restart the router (power off 20 seconds) and the PC.

Why this works: it clears bad cached credentials and forces a clean handshake with the router.

Check quickly
  • Network name and password confirmed (no hidden caps/extra spaces).
  • Router shows internet light and other devices can browse.
  • You're close to the router (try within one room for the test).
Visual checklist
1) Forget network
Settings » Network & Internet » Wi‑Fi » Manage known networks » Forget.
2) Airplane toggle
Quick Settings: Airplane mode on, wait 5 seconds, then off.
3) Rejoin Wi‑Fi
Select the SSID, re‑enter the password carefully, then connect.
4) Restart gear
Router off for 20 seconds, then on. Restart the PC and retry.
Handy script
ipconfig /release
ipconfig /renew
ipconfig /flushdns

Run from Command Prompt (Admin) if the reconnect still stalls.
